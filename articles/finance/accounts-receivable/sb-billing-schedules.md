# <a name="billing-schedule-overview"></a>計費排程概觀

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

在 **計費排程** 頁面，可以建立、刪除和編輯計費排程。 還可以查看計費排程清單。 建立計費排程時，其預設值由與其關聯的帳單群組確定。 其他資料在 **定期合約計費參數** 頁面中設定。

## <a name="create-a-billing-schedule"></a>建立計費排程

若要建立計費排程，請按照以下步驟操作。

1. 在 **計費排程** 頁面中，選取 **新建**。
2. 在 **建立計費排程** 對話方塊的 **計費排程群組** 欄位中，選擇計費排程群組。
3. 在 **客戶帳戶** 欄位中，選擇客戶帳戶。
4. 在 **開始日期** 欄位中，選擇開始日期，然後在 **週期數** 欄位中輸入週期數。 **結束日期** 欄位會根據輸入的週期數進行更新。 如果您更新 **帳單結束日期** 欄位，**週期數** 欄位將更新為 **0** (零)。
5. 選取 **確定**。
6. 在 **計費排程** 頁面內 **一般** 索引標籤的 **說明** 欄位中，輸入計費排程的說明。
7. 在 **里程碑範本** 欄位中，選擇 **里程碑計費** 的里程碑範本。

**發票帳戶** 和 **貨幣代碼** 等欄位會根據客戶資料更新。

**計費頻率** 和 **計費間隔** 欄位將根據所選的計費排程群組自動設定。

8. 若要建立單獨的發票，將 **單獨開票** 選項設為 **是**。
9. 若要在最終計費週期後自動續約計費排程，則將 **自動續約** 選項設定為 **是**，然後設定 **按續約新增明細** 欄位。

**參數** 欄位將根據 **定期合約計費參數** 頁面中的值自動設定。

10. 若要按比例分配計費排程的金額，將 **按比例分配部分週期** 選項設定為 **是**。
11. 若要將計費排程詳細資料明細保持與月底一致，將 **與月份對齊** 選項設定為 **是**。
12. 在 **合約開始日期** 和 **合約結束日期** 欄位中，輸入合約的開始日期和結束日期。 這些日期僅用於提供資料。

**付款** 欄位顯示來自客戶的客戶付款資料。 當明細項目處於暫停或終止狀態時，無法更改付款資料。

> [!NOTE]
> 當您按項目合併發票時，**付款條款**、**方式** 和 **計費排程** 欄位的值必須相符。 否則，無法合併發票。

13. 在 **地址** 索引標籤，您可以查看和更新 **交貨地址** 和 **付款人地址** 欄位。
14. 在 **連絡資料** 索引標籤中，您可以將終端使用者帳戶與計費排程相關聯。
15. 在 **連絡資料** 欄位中，可以輸入連絡人、電子郵件地址和 Internet 位址。
16. 若要追蹤合作夥伴佣金資料，則設定 **合作夥伴帳戶** 和 **合作夥伴佣金比率** 欄位。 這些欄位僅用於提供資料。
17. 在 **總計** 索引標籤中，可以查看已為計費排程計算的各種總計。
18. 在 **暫停** 索引標籤中，可以查看有關在取消暫停後何時暫停計費排程的稽核資料。
19. 在 **終止** 索引標籤中，可以查看已套用到計費排程或從計費排程中刪除的終止歷史記錄。
20. 選取 **儲存**。
21. 在 **計費排程明細** FastTab，選擇 **新增明細**。
22. 在 **項目編號** 欄位中，選擇項目編號。 如果加入的項目是收入拆分中的父系項目，則這些明細會自動更新為子項目。 您只能編輯收入拆分中的父系項目。 然後所有子項目會相應更新。
23. 在 **項目類型** 欄位中，選擇項目類型。
24. 更新開始日期和結束日期。
25. 在建立發票之前，您可以透過更新 **計費頻率** 欄位來更改計費頻率。 建立計費排程的第一張發票後，將無法更改計費頻率。
26. 在 **單位** 欄位中，選擇項目的計量單位。
27. 在 **定價方法** 欄位中，選擇項目的定價方法。

**單價** 欄位將根據庫存自動設定。 但是，如果您將定價方法更改為 **統一**，則可以對其進行更新。

## <a name="remove-a-line-item"></a>移除明細項目

若要從計費排程中刪除項目，請按照以下步驟操作。

1. 在 **計費計畫** 頁面的 **計畫編號** 欄位中，選擇要編輯的計費排程編號。
2. 在 **計費排程明細** FastTab 中，選擇要刪除的明細，然後選擇 **移除**。
3. 選取 **儲存**。

本主題的其餘部分介紹可用於 **計費排程明細** FastTab 的明細操作和詳細資料。

## <a name="billing-schedule-line-actions"></a>計費排程明細操作

**計費排程明細** FastTab 上的按鈕可操作各個項目。

| 按鈕 | 描述 |
|--------|-------------|
| 新增細項 | 在計費排程中新增細項。 |
| 從項目清單中加入 | 透過在清單中選擇多個項目，將它們加入計費排程中。 |
| 移除 | <p>從計費排程中移除所選的明細。</p><p>對於屬於收入拆分的項目，您只能刪除父系項目。 然後自動刪除所有關聯的子項目。</p> |
| 查看計費詳細資料 | 查看計費詳細資料。 |
| 終止 | <p>終止所選的明細。 僅當所選明細的狀態為 **使用中** 時，才可使用此按鈕。</p><p>對於屬於收入拆分的項目，您只能終止父系項目。</p> |
| 取消終止 | <p>取消所選明細的終止。 僅當所選明細的狀態為 **終止** 時，才可使用此按鈕。</p><p>對於屬於收入拆分的項目，您只能取消父系項目的終止。</p> |
| 暫停 | <p>選擇將所選明細置於暫停狀態的詳細資料。</p><p>對於屬於收入拆分的項目，您只能暫停父系項目。</p> |
| 取消暫停 | <p>取消所選明細的暫停。</p><p>對於屬於收入拆分的項目，您只能取消父系項目的暫停。</p> |
| 升級和折扣 | 此按鈕不適用屬於收入拆分中的項目，但收入拆分使用 **零金額** 分配方法的父系項目除外。 |
| 延期 | <p>輸入所選線路的延期選項。</p><p>此按鈕不適用於收入拆分中的父系項目。</p> |
| 里程碑分配 | <p>查看和更新選定明細的里程碑分配資料。</p><p>僅當計費排程明細項目是里程碑項目時，才可使用此按鈕。</p> |
| 支援和續約 | <p>查看和更新選定明細的支援和續約資料。</p><p>僅當計費排程明細項目是支援和續約項目時，才可使用此按鈕。</p> |
| 顯示維度 | 顯示或隱藏出現在 **計費排程明細** 網格中的維度列。 |
| 計算單價 | <p>計算項目的單價，以便客戶可以分期支付合約金額 (例如，每天、每月、每季、每半年或每年)。 您可以選擇合約價格和價格頻率。</p><p>您可以查看更改的稽核線索：舊合約價格和頻率、新合約價格和頻率、進行更改的使用者以及更改的日期和時間。</p> |
| 對齊日期 | <p>指定續約項目的對齊日期。</p><p>如果您在 **項目群組** 欄位中，所有項目都使用計費排程內項目群組中第一個項目的對齊日期。 如果 **項目群組** 欄位為空白，您可以指定用於所有項目的對齊日期。</p><p>僅在 **計費頻率** 欄位設定為 **每年** 時，才可使用此按鈕。</p> |
| 未開帳單收入 | <p>將項目設定為使用未開帳單收入功能。 然後，您可以指定項目的未開帳單收入和未開帳單折扣帳戶。</p><p>僅項目在 **項目類型** 欄位設定為 **標準** 時，才可使用此按鈕。 不適用於現有計費排程或已建立發票的計費排程。</p> |
| 加入收入拆分子項 | <p>選擇要加入銷售訂單的子項目。</p><p>此按鈕僅適用於收入拆分中的父系項目。</p> |
| 預先定價選項 | 編輯項目的定價選項。 |

## <a name="billing-schedule-line-details"></a>計費排程明細詳細資料

當您在 **計費排程明細** FastTab 中選取明細時，您可以查看該明細的具體詳細資料。 這些詳細資料分別存在不同的索引標籤。

### <a name="general-tab"></a>一般索引標籤

**一般** 索引標籤提供以下資料。

| 欄位或區段 | 描述 |
|------------------|-------------|
| 使用 | <p>本區段提供有關使用項目的資料。 包含以下欄位：</p><ul><li>**使用識別項** – 計量或使用項目的識別項。</li><li>**讀取選項** – 使用讀取選項：**讀取** 或 **消耗**。</li><li>**估計消耗量** – 指定具有未建立發票期間的使用項目的估計消耗量。 在 **計費詳細資料** 頁面中，您可以查看估計消耗的計費詳細資料明細。</li></ul> |
| 外部參考\* | 本區段包含 **外部** 和 **明細編號** 欄位，您可以在其中指定外部參考資料。 |
| 里程碑 | <p>本區段提供有關里程碑項目的資料。 包含 **估計完成日期** 欄位，顯示項目完成日期。</li></ul> |
| 文字 | 該明細的註釋。 文字將翻譯為客戶或法律實體的預設語言。 |
| 項目群組 | 明細項目的項目群組。 |
| 對齊日期 | 計費排程的對齊日期。 |

\*當您在 **產生發票** 頁面中按項目合併發票時，**外部參考** 欄位必須相符。 即使一個字元不同，這些項目就不會合併到發票上。 **外部參考** 區段中的欄位均不經驗證檢查，**明細編號** 欄位中的值必須為正整數。

### <a name="address-tab"></a>地址索引標籤

**地址** 索引標籤提供以下資料。

| 欄位 | 描述 |
|-------|-------------|
| 交貨地址 | <p>選擇明細項目的交貨地址。 預設交貨地址是來自 **地址** FastTab 中的主要交貨地址。</p><p>更改地址時，可以選擇以下地址選項：</p><ul><li>**地址** – 選擇目前客戶的地址。</li><li>**使用中** – 選擇目前客戶使用中的地址。</li><li>**其他地址** – 選擇任何客戶記錄的地址。</li></ul><p>對於使用收入拆分的項目，只能編輯父系項目的地址。 子項目的地址與父系項目的地址一致，不能單獨編輯。</p> |
| 付款人地址 | <p>選擇明細項目的付款人地址。 預設交貨地址是來自 **地址** FastTab 中的主要交貨地址。 您可以根據可用地址的用途，視需要更改地址：</p><ul><li>如果沒有地址具有 **發票** 用途，預設的付款人地址將是客戶的主要地址，而不管其用途如何。</li><li>如果一個或多個地址的用途是 **發票**，預設的付款人地址是最近輸入的地址。</li><li>如果一個或多個地址的用途為 **發票**，並且其中一個發票地址被設定為主要地址，則預設的付款人地址為具有 **發票** 用途並被設定為主要地址的地址。</li><li>對於使用收入拆分的項目，只能編輯父系項目的地址。 子項目的地址與父系項目的地址一致，不能單獨編輯。</li></ul> |

### <a name="product-tab"></a>產品索引標籤

**產品** 索引標籤提供以下資料。

| 欄位或區段 | 描述 |
|------------------|-------------|
| 儲存尺寸 | <p>此區段顯示項目的儲存資料。 包含 **序號** 欄位，其顯示項目的序號。</p><p>在支援和續約過程期間，從初始銷售訂單中複製此序號。 對於使用收入拆分的項目，父系項目的序號將複製到所有子項目。 當 **定期合約計費參數** 頁面上的 **複製序號** 選項設定為 **是** 時，將複製序號。</li></ul> |
| 產品尺寸 | 根據為計費排程明細選擇的 **變型編號** 值，會自動更新項目的產品詳細資料和值。 |

### <a name="account-tab"></a>帳戶索引標籤

| 欄位 | 描述 |
|-------|-------------|
| 主科目 | 在銷售明細上建立的主科目。 預設值來自銷售訂單。 此欄位可以保持空白。 |
| 項目財務維度 | <p>根據客戶和項目記錄的預設財務維度值。</p><p>對於使用收入拆分的項目，如前所述，子項目使用客戶和項目記錄的財務維度值。 如果您必須更新子項目的財務維度值，可以匯入資料實體。</p> |

### <a name="renewals-tab"></a>續約索引標籤

**續約** 索引標籤提供以下資料。

| 欄位 | 描述 |
|-------|-------------|
| 自動續約 | <p>該值指示計費排程明細是否自動續約下一個計費週期：</p><ul><li>**是** – 當建立發票時，會自動續約下一個計費週期。</li><li>**否** – 不自動續約計費排程明細。 必須手動更新計費排程。</li></ul> |
| 每次續約要加入的明細 | 要加入計費排程續約的明細數量。 |

此外，**續約** 索引標籤提供以下按鈕。

| 按鈕 | 描述 |
|--------|-------------|
| 未開帳單收入日記帳分錄稽核 | 查看使用未開帳單收入功能的項目之所有更改。 |
| 加入續約期限 | 為項目加入續約期限。 新續約期限的開始日期是上一期限結束日期之後的隔天。 **續約結束日期**、**延期開始日期**、**延期結束日期**、**項目數量**，和 **單價** 欄位可以更新。 |
| 修改續約期限 | <p>修改續約期限。 對於初始期限，您可以在建立初始日記帳分錄之前更改延期開始和結束日期。 對於後續期限，無法更改開始日期。 開始日期永遠是上一個期限結束之後的隔天。</p><p>如果在修改的期限之後有續約期限，則無法更改該期限的日期。 在這種情況下，只能更新續約項目的 **數量** 和 **單價** 欄位。</p><p>例如，有三個期限。 <ul><li>無法更改第一個期限，因為它已經開始。</li><li>對於第二個期限，只能更改數量和單價。</li><li>對於第三個期限，可以更改除了開始日期之外的所有值。 此外，**從範本排程** 選項可讓您根據未開帳單收入項目的範本建立延期排程。 當此選項設定為 **是** 時，在 **範本** 欄位中選擇延期範本，然後視需要更改延期開始和結束日期。 後續的續約期限使用相同的延期範本。 不過，延期範本可以更改。</p></li></ul> |

### <a name="termination-tab"></a>終止索引標籤

**終止** 索引標籤提供以下資料。

| 欄位 | 描述 |
|-------|-------------|
| 終止日期 | 計費排程行終止的日期。 預設值來自標題上的 **終止日期** 欄位。 您可以根據要求更改此值。 |
| 終止類型 | 終止類型。 預設值來自標題上的 **終止類型** 欄位。 |

### <a name="hold-tab"></a>暫停索引標籤

如果使用收入和費用延期，**暫停** 索引標籤顯示延期日期。

### <a name="escalation-and-discount-tab"></a>升級和折扣索引標籤

**升級和折扣** 索引標籤提供以下資料。

| 欄位 | 描述 |
|-------|-------------|
| 升級 | <p>選擇是否允許升級計費排程明細。 建立計費排程明細時，將應用標題中的任何升級明細。</p><ul><li>**是** – 可以對明細升級。 選擇此選項後，可以在 **升級和折扣** 頁面設定計費排程行的升級。</li><li>**否** – 不可以對明細升級。</li></ul><p>預設設定是根據所選 **計費排程組** 而定。</p> |

### <a name="price-changes-tab"></a>價格更改索引標籤

對於從 **標準** 價格更改為 **統一** 價格的明細，**價格更改** 索引標籤中的網格包括以下列：

- 更改日期
- 由使用者更改
- 標準價格
- 統一價格
- 價格更新
