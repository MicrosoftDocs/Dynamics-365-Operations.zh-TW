# <a name="len-er-function"></a>LEN ER 函數

[!include [banner](../includes/banner.md)]

`LEN` 函數將指定字串中的字元數作為 *整數* 值回傳。

## <a name="syntax"></a>語法

```vb
LEN (text)
```

## <a name="arguments"></a>引數

`text`：*字串*

指定文字的 *字串* 值。

## <a name="return-values"></a>回傳值

*整數*

產生的數值。

## <a name="example"></a>範例

`LEN ("Sample")` 會回傳 **6**。
