# <a name="file-destination"></a>檔案目的地

[!include [banner](../includes/banner.md)]

您可以為設定為產生輸出文件的電子報表 (ER) 格式的每個 FOLDER 或 FILE 組件設定檔案目的地。 根據目的地的設定，產生的文件可從網路瀏覽器下載。

在 **目的地設定** 頁面上，如果將 **啟用** 設定為 **是**，則在設定完成執行後會打開一個打開或儲存對話方塊。
