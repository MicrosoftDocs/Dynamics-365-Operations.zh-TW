# <a name="create-configuration-providers-and-mark-them-as-active"></a>建立設定提供者並將其標記為活動

[!include [banner](../../includes/banner.md)]

本主題說明指派給系統管理員或電子報表開發人員角色的使用者如何為電子報表 (ER) 建立設定提供者。 每個 ER 設定都將提供者稱為設定的作者。 在此範例中，您將為樣本公司 Litware, Inc. 建立一個設定提供者。這些步驟可以在任何公司中執行，因為 ER 設定提供者在所有公司之間共用。

## <a name="create-a-provider"></a>建立一個提供者
1. 進入左上角的 **瀏覽窗格** 並選取 **組織管理**。
2. 進入 **工作區 > 電子報表**。
3. 進入 **相關連結 > 設定提供者**。
4. 選取 **新建**。
    - 提供者記錄具有唯一的名稱和 URL。 如果 Litware, Inc. (https://www.litware.com) 的記錄已存在，請查看此頁面的內容並跳過此程序。  
5. 在名稱欄位，輸入 `Litware, Inc.`。
6. 請在網址欄位，輸入 `https://www.litware.com`。
7. 選取 **儲存**。
8. 關閉頁面。

## <a name="select-as-an-active-provider"></a>選取作為有效的提供者
1. 選取 Litware, Inc. 提供者。
2. 選取 **設定為作用中**。
