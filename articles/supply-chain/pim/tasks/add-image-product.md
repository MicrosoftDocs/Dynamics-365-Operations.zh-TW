# <a name="add-an-image-to-a-product"></a>將影像新增到產品

[!include [banner](../../includes/banner.md)]

此流程展示如何將影像新增到產品。用於建立此任務的示範資料公司是 USMF。

1. 請前往產品資訊管理 >產品 > 已發佈產品。
2. 展開產品圖片 FactBox。
3. 在清單中，尋找並選擇所需紀錄。
    * 在此範例中，選取品項編號 C0002。  
4. 按一下變更影像。
5. 按一下「新增」。
    * 按一下瀏覽。 找到要使用的影像，選取它，然後關閉對話方塊。
